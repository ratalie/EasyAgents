## Espanol

### Quien eres en este contexto?

Eres alguien que ayuda a otra persona a usar tecnologia. Puede ser:
- Un familiar que ayuda a un adulto mayor
- Un cuidador de una persona con discapacidad
- Un profesor o facilitador comunitario
- Un voluntario en un programa de inclusion digital

### Que puedes hacer con este proyecto?

Puedes crear asistentes de IA personalizados para las personas que apoyas.
Por ejemplo:
- Un asistente extra paciente para una persona mayor que quiere usar internet
- Un asistente con lenguaje simple para alguien aprendiendo a leer
- Un asistente que usa lector de pantalla para una persona con discapacidad visual

### Como configurar un asistente para alguien

1. Ve a la pagina web del wizard
2. Responde las preguntas PENSANDO en la persona que lo va a usar
3. En la pregunta de accesibilidad, selecciona lo que aplique
4. Genera el prompt
5. Configuralo en el dispositivo de la persona

### Tips para facilitadores

- Prueba el asistente tu primero antes de darselo a la persona
- Sientate con la persona las primeras veces que lo use
- Dile que no hay preguntas tontas
- Si el asistente no funciona bien, ajusta el prompt y prueba de nuevo
- Recuerda: la meta es que la persona sea cada vez mas independiente

### Modos de accesibilidad recomendados por situacion

| Situacion | Modos recomendados |
|---|---|
| Persona mayor sin experiencia digital | Paso a paso + Lenguaje simple |
| Persona con discapacidad visual | Lector de pantalla + Paso a paso |
| Persona con dificultad de aprendizaje | Lenguaje simple + Anti-sobrecarga |
| Persona con TDAH | Anti-sobrecarga |
| Persona con ansiedad tecnologica | Paso a paso + Anti-sobrecarga |
| Nino/a aprendiendo | Lenguaje simple |

## English

### Who are you in this context?

You're someone who helps another person use technology. You might be:
- A family member helping an older adult
- A caregiver for a person with a disability
- A teacher or community facilitator
- A volunteer in a digital inclusion program

### What can you do with this project?

You can create personalized AI assistants for the people you support.
For example:
- An extra patient assistant for an older person who wants to use the internet
- A simple language assistant for someone learning to read
- A screen reader assistant for a person with visual disability

### How to set up an assistant for someone

1. Go to the wizard web page
2. Answer the questions THINKING about the person who will use it
3. On the accessibility question, select what applies
4. Generate the prompt
5. Set it up on the person's device

### Tips for facilitators

- Test the assistant yourself first before giving it to the person
- Sit with the person the first few times they use it
- Tell them there are no stupid questions
- If the assistant doesn't work well, adjust the prompt and try again
- Remember: the goal is for the person to become increasingly independent

### Recommended accessibility modes by situation

| Situation | Recommended modes |
|---|---|
| Older person without digital experience | Step by step + Simple language |
| Person with visual disability | Screen reader + Step by step |
| Person with learning difficulty | Simple language + Anti-overwhelm |
| Person with ADHD | Anti-overwhelm |
| Person with technology anxiety | Step by step + Anti-overwhelm |
| Child learning | Simple language |
