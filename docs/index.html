Crea tu propio asistente de IA

Sin programar. Sin pagar. Para todos.

¿Qué es esto?

Esta herramienta te ayuda a crear un asistente virtual personalizado. Tú respondes unas preguntas simples y nosotros generamos las instrucciones listas para usar.

Después copias esas instrucciones en cualquier chat de IA gratuito (Claude, ChatGPT, Gemini) y listo: tienes tu propio asistente.

Gratis

No necesitas pagar nada. Usa tu cuenta gratuita de cualquier IA.

Sin código

Solo responde preguntas. Nosotros generamos todo por ti.

Accesible

Diseñado para que cualquier persona pueda usarlo, sin importar su experiencia.

Paso 1 de 6

¿Para quién es tu asistente?

Primero dime: ¿quién va a usar este asistente?

¿Qué problema quieres resolver?

Dilo como se lo dirías a un amigo. No necesitas palabras técnicas.

Ejemplos: "Quiero algo que responda preguntas de mis clientes" / "Necesito ayuda para organizar mi dia" / "Quiero algo que me ayude a estudiar"

¿Cómo quieres que se comunique?

Elige la personalidad de tu asistente.

¿Hay alguna necesidad especial?

Puedes elegir varias opciones. Esto adapta tu asistente para que sea más accesible.

¿Hay algo que NO debe hacer?

Define límites para tu asistente. Si no se te ocurre nada, puedes dejarlo vacío.

Ejemplos: "No dar consejos medicos" / "No hablar de politica" / "No compartir datos personales"

¡Tu asistente está listo!

Revisa el resumen. Si quieres cambiar algo, usa el botón "Atrás".

¡Aquí está tu asistente!

Copia el texto de abajo y pégalo como primer mensaje en tu chat de IA favorito.

¡Copiado!
